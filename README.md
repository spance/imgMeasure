<h1 align="center">imgMeasure</h1>

<p align="center">
  <strong>零依赖 · 单文件 · 开箱即用的工程图像测量标注工具</strong>
</p>

<p align="center">

  整个工具是一个 HTML 文件，双击打开即用，无需安装、构建或部署。
</p>

---

## 特性

- **🖼 图像浏览** — 滚轮缩放（以鼠标为中心）、平移拖拽、自适应画布、多图层叠加与透明度控制
- **📍 点标记** — 单击标注，实时显示坐标
- **📦 矩形框** — 拖拽绘制，8 个控制点自由调整尺寸；支持对角线坐标输入与 XYWH 输入
- **📏 线段测距** — 两点连线，自动计算并显示像素距离
- **⭕ 圆测量** — 圆心 + 边缘两点，自动显示半径、面积、周长
- **🔄 正反向交互** — 鼠标标注 → 坐标数据，坐标数据 → 画布标注；支持 `[x,y,w,h]` 数组格式批量输入
- **🗂 多图层** — 图层可叠加、拖拽排序、独立透明度与可见性控制
- **↩️ 撤销/重做** — 全操作链 Undo / Redo，快捷键 Ctrl+Z / Ctrl+Shift+Z
- **✂️ 区域导出** — 选中矩形区域可一键裁切导出为 PNG
- **🎨 深色主题** — 护眼深色 UI，IDE 风格可折叠面板布局

## 使用说明

### 上传图片

- 拖拽图片到画布区域
- 点击菜单 `文件 → 打开图片`
- 支持叠加多张图片为独立图层

### 标注工具

| 快捷键 | 工具 | 操作方式 |
|--------|------|---------|
| `V` | 选择 | 点击标注选中，拖拽移动，点击空白取消选中 |
| `P` | 点标记 | 单击放置坐标点 |
| `R` | 矩形 | 按住拖拽绘制矩形框 |
| `L` | 线段 | 点击起点 → 点击终点，自动显示距离 |
| `C` | 圆 | 点击圆心 → 点击边缘，自动显示半径/面积 |
| `H` / `Space` | 平移 | 按住拖拽画布 |

### 其他快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
| `Delete` | 删除选中标注 |
| `+` / `-` | 缩放 |
| `Ctrl+0` | 适配画布 |

### 反向输入

右侧面板「反向输入」支持直接输入坐标值生成标注：

- **点标记**: `X, Y`
- **矩形**: `X, Y, W, H`
- **矩形(对角线)**: `X1, Y1, X2, Y2`
- **线段**: `X1, Y1, X2, Y2`
- **圆**: `CX, CY, R`

支持数组格式批量输入，如 `[10, 20, 80, 60]`，也支持 `10,20,80,60` 或空格分隔。

### 图层管理

- 点击图层切换活跃图层
- 拖拽图层调整叠加顺序
- 滑块调整透明度，眼睛图标切换可见性
- 支持新建多个标注层

### 区域导出

选中矩形标注后，属性面板出现「导出裁切区域」按钮，点击即可将矩形区域内的图片内容导出为 PNG 文件。

## 架构

```
┌──────────────────────────────────────────────────┐
│  菜单栏  文件 | 编辑 | 视图 | 工具 | 帮助         │
├─────────┬────────────────────────┬────────────────┤
│ 左面板   │  Canvas (图像渲染)     │ 右面板          │
│ ·图层树  │  SVG   (标注叠加)      │ ·属性编辑       │
│ ·标注列表│                        │ ·反向输入       │
├─────────┴────────────────────────┴────────────────┤
│  状态栏  工具 | 坐标 | 缩放 | 图片尺寸              │
└──────────────────────────────────────────────────┘
```

**渲染架构：** Canvas + SVG 混合。Canvas 负责图像的高性能缩放平移渲染，SVG 叠加层负责标注元素的交互（选中、拖拽、控制点），两者通过统一坐标变换矩阵同步。

**模块设计（单文件内）：**

| 模块 | 职责 |
|------|------|
| EventBus | 发布/订阅事件总线，解耦模块间通信 |
| StateManager | 集中状态管理 + Undo/Redo 命令栈 |
| Viewport | Canvas 渲染、坐标变换、缩放平移 |
| LayerManager | 图层 CRUD、排序、透明度 |
| ToolManager | 工具注册/切换、鼠标/键盘事件分发 |
| AnnotationRenderer | SVG 标注渲染、控制点、标签 |
| UIManager | 面板、菜单、状态栏 DOM 操作 |

## 技术栈

纯原生实现，零外部依赖：

- **HTML5 Canvas** — 图像渲染
- **SVG** — 标注交互层
- **ES6 Class** — 模块化架构
- **CSS Custom Properties** — 主题系统
